# preJS minimum template

preJSプロジェクトのミニマムテンプレート

## インストール

```bash
git clone git@github.com:shimizu/prejs-minimum-template.git
cd prejs-minimum-template
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## プレビュー

```bash
npm run preview
```

## デプロイ

```bash
npm run deploy
```
